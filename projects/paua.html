Pāua Dashboard

Using fake data

The Pāua dashboard was an already established website built by Dragonfly Science for the Pāua Industry Council. I was asked to create a new dashboard for users based on wireframes that provided a summary of the data available to them.

This project was really enjoyable to me as the website was written in Elm, a compile-to-Javascript functional language. I loved the challenge of learning and having to start writing code in a completely new language within a couple of days.

The data had to be displayed conditionally depending on the user's role, and what area and year were selected. The data visualisation and processing necessary to make the information accessible was interesting and something I'd love to do more of in future.
