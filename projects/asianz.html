Asia NZ Foundation Media Centre

Live site

The Asia New Zealand Foundation asked Salted Herring to design a website for their Media Centre, and I was contracted to do the front end development for the project. The CMS was written in Silverstripe, and so the front end was templated HTML5, using BEM SCSS for styling and Javascript/jQuery.

This project was highly organised, with regularly updated Trello and kanban boards. We were working in a small team to a tight deadline, and there was consistent communication with the designers, back end developers and project management. Fast coding was a priority.

I enjoyed writing design-focused front end code, and delving a bit into Silverstripe. Keeping every aspect of the code organised, readable and maintainable was a must, and seeing how other developers structure their projects was interesting and valuable.
