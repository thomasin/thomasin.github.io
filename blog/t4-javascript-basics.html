How does Javascript compare to HTML/CSS?

HTML/CSS are the building blocks of a website design, they provide a solid base. Javascript brings a design the ability to be dynamic, interact and respond to what the user is doing. They go hand in hand with each other but bring different functions to a design.

Control flow and loops

Control flow and loops can be related to the simple process of eating. The control flow is all the decisions you make when deciding what to eat (if... I have this in the pantry, else... I can make that instead). A loop is simply an action that continues until you tell it stop (eating until your stomach decides you're full up). All these minute decisions and processes chain together to form a complex and tailored system. Control flow, like this, is the series of if... elses that make up a Javascript process.

Accessing data in arrays vs objects

Arrays and objects do seem very similar. A big difference between them is that data in an array can be referenced using it's index number (0, 1, 2...) and so is accessed numerically. Objects on the other hand are referenced using keys, usually a word. This places importance on the key, whereas in arrays the number associated with a value is usually far less important than the value itself.

What functions are, and why they're useful

Functions are a block of code that take a value, do something with it and return another value. They are useful because they are essentially outside of the flow, only used when called upon. Functions reduce repitition by being able to be used more than once in different situations, which can simplify code a lot. They are the building blocks of Javascript and massively useful.
