Symbols

  • (): Brackets are used to send and receive values, whether variable names, numbers or class names function hello(n), .add("n").
    They can be used around conditionals if(n) and when manipulating numbers 1 + (n*2)
  • []: Square brackets are used in conjunction with arrays, to define an array or select a value using an index number array = [n], array[0] = n
  • {}: Curly brackets are used to define the start and the end of function calls, loops, if statements and other blocks of code. They can be nested to define smaller blocks within those. for(n) {n}, function hello(n) {} They're also used to define objects var obj = {i'm an object}
  • '', "": Single and double quotes are used to define strings var n = "string". They can be used also around selectors and identifiers that are not functions ("click", function()). There is no differene between them, except if you use one kind you can use the other within the quote and it will not break the string, for example "it's a good day" will display correctly but 'it's a good day' won't work.
