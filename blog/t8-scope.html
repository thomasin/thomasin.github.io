What 'scope' is, and how it works in Javascript

Scope is how much of the rest of the code a certain function can see or a variable be used. For example, if a variable is defined outside of a function in the global scope it can be used throughout the whole file. If it is defined with a function it can only be seen within the function, the local scope. Every time you create a new function even if it's nested it creates a new local scope within it like a little bubble that you can put variables in you don't need in the outside code.
